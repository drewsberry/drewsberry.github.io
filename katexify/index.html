katexify - a KaTeX Jekyll plugin

07 Oct 2014
‖
Coding

If you follow Hacker News, then you might’ve seen the latest development from Khan Academy: KATEX.

KATEX is a replacement for the very popular and very slow MathJax, which although it has not fully implemented all the different symbols, promises to serve as a fully featured alternative.

This post presents a Jekyll plugin to render all your LATEX equations when you run jekyll build, so that all you need to do is bundle the KATEX CSS with your site and you’ve got lightning fast HTML mathematics.

So why should you use KATEX? Well, here’s a few reasons:

  • It’s super fast;
  • It supports server side rendering;
  • Pure CSS (and JS if you’re rendering client-side) - no external dependencies;
  • It’s super fast.
